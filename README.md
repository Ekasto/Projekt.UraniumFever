# Uranium Fever

A strategic board game about resource management, trading, and survival in a world facing natural disasters.

## Game Overview

Uranium Fever is a resource management and trading game where players compete to deliver resources to their bases while dealing with natural disasters and strategic challenges.

### Core Mechanics

- **3 Game Boards**: Three separate 8x8 boards with different starting bases
- **Resource Management**: Draw resource cards each round and deliver them to matching bases
- **Trading System**: Exchange mismatched resources with other players
- **Disaster Events**: Random catastrophes that affect gameplay after round 3

### Resources

- Electricity (Strom)
- Food (Nahrung)
- Medicine (Medizin)
- Player's Choice

### Building Types

- Defense structures
- Factories
- Research facilities
- Cars / Destroyed Cars
- Bridges
- Highways

### Disasters

- **Earthquake**: Destroys bridges
- **Flood**: Blocks areas
- **Tornado**: Destroys houses
- **Thief**: Steals resources
- **Donkey Card**: All players roll dice for building destruction
  - Roll 1-2: One building destroyed
  - Roll 3-4: Two buildings destroyed
  - Roll 5-6: Three buildings destroyed

### Win/Lose Conditions

If a city falls, all resources are lost.
